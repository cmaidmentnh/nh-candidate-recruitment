How It Works
  1. Looks up towns in each selected district
  2. Queries voter checklist for registered Republicans in those towns
  3. Cross-references against existing FEC-sourced prospects
  4. Enriches unmatched prospects with voter data
  5. Adds voter profile signal (+10 score) for cross-matches

This scan does NOT create prospects from bare voter records alone (too many results). It's designed to enrich and validate prospects found through other sources like FEC donations.
